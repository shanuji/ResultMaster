# ResultMaster

ResultMaster is an offline school result management Android app built with Flutter. Sprint 1 establishes the project foundation, Material 3 Excel-inspired UI shell, Riverpod state management, SQLite database layer, and navigation structure.

## Tech Stack

- Flutter
- Riverpod
- sqflite SQLite storage
- Material Design 3
- Clean Architecture-ready folder structure

## Sprint 1 Scope

Business logic is intentionally not implemented yet. The current app provides a dashboard shell, dynamic-subject-tab preparation, database bootstrap services, and feature folders for future result-management workflows.
